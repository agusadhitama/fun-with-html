# FunWithHTML 🎨💻

> 🎉 Project HTML + CSS + JS interaktif buat pemula belajar coding dengan cara yang fun!  
> Klik tombol untuk pesan acak dan background berubah warna 🌈✨

---

Live Demo 🌐  
https://agusadhitama.github.io/fun-with-html/

---

## Fitur Utama
- 🌟 Responsive untuk HP & tablet  
- 🎯 Interaktif: tombol klik → pesan acak + background berubah  
- 🧩 Beginner-friendly: mudah dimodifikasi  
- 🗂️ Struktur sederhana: `index.html` | `style.css` | `script.js`  

---

## Cara Pakai
```bash
# Clone repo
git clone https://github.com/username/FunWithHTML.git

# Buka folder project di VS Code
# Install extension Live Server
# Klik kanan index.html → Open with Live Server
# Klik tombol dan lihat background + pesan berubah!
